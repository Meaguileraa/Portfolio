## Installation

### Virtual Environment
Create a virtual environment to install requirements 

```sh
$ virtualenv
$ source env/bin/activate
```

### Prerequisites
All the prerequisites are in the requirements.txt file 

```sh
pip3 install -r requirements.txt
```

### Run Server 
```sh
python3 server.py
```
